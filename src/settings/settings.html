MergeChain

GitLab-style merge dependencies for GitHub pull requests.

Your credentials never leave this browser
  • The access token is stored in chrome.storage.local on this device. Never synced to the cloud, never uploaded.
  • Sent only to api.github.com over HTTPS, from the extension's background worker.
  • No backend server exists. Sign-in uses GitHub's device flow with a public client ID and no secret, so there is nowhere else for anything to go. Verify in chrome://extensions → Details → site access: only github.com and api.github.com.
  • The GitHub page you browse never receives the token; only the isolated background worker holds it.
